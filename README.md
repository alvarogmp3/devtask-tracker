# DevTask Tracker - Monolito Artesanal

Proyecto FullStack desarrollado con Vanilla JS y Node.js para la gestión de tareas técnicas.

## Instalación y Configuración

1. **Clonar el repositorio:** `git clone https://github.com/alvarogmp3/devtask-tracker.git`
2. **Backend:**
   - Ir a la carpeta `/backend`.
   - Ejecutar `npm install` para instalar las dependencias (Express, Mongoose, Dotenv, Cors).
   - Crear un archivo `.env` con la variable `MONGO_URI`.
   - Arrancar con `node server.js`.
3. **Frontend:**
   - Abrir `index.html` en el navegador.

## Requisitos cumplidos
- API REST con Node.js y Express.
- Persistencia en MongoDB Atlas con esquemas Mongoose.
- Interfaz dinámica con Vanilla JS y Fetch API.
